# Discuss

A newer version of the application built in the [Elixir course taught by Steve Grieder](https://www.udemy.com/share/1021ea3@eM5Sw7FHByU3TL3O-YwM0ibP7MEEvPKwZWayZ_MeJ2QWKOnnaIoGFDtAXiWfxQz55g==/). Wrote a Medium article to break down the processes involved in building a full web application with Elixir Phoenix with the following functionalities
- OAuth Authentication
- Full-stack Development with Phoenix & HEEx templates (No LiveView)
- Real-time update with Phoenix Channels
- And Many More...

You can check out the [link to the Medium article](https://medium.com/@nature.exs/phoenix-1-7-cb5c5b675ec4) and refer back to this repo if you feel lost at any time

You want to jump on the wagon of Phoenix and experience the awesomeness of Elixir in web development; this is a great place to start and master its fundamentals.

Feel free to drop your questions or comments if you spot any issues

===========================================================================

## How to Run locally

  * Run `mix setup` to install and setup dependencies
  * Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
